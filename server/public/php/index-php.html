üöÄ Quick Setup Guide

Download Files

Download index.html dan backend.php dari daftar di atas.

Edit index.html

Buka file, cari baris ~231, ganti:
data-sitekey="YOUR_SITEKEY_HERE"
dengan sitekey Anda dari dashboard.

Edit backend.php

Buka file, cari baris ~27, ganti:
define('API_SECRET_KEY', 'YOUR_SECRET_KEY_HERE');
dengan secret key Anda.

Upload ke Server

Upload kedua file ke server Anda:
/var/www/html/php/index.html
/var/www/html/php/backend.php

Test!

Akses di browser:
https://api-server.cloud/php/index.html
